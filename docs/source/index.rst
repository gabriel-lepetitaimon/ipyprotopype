
ipyprotopypes
=====================================

Version: |release|

Custom Jupyter Widgets for ProtoPypes.


Quickstart
----------

To get started with ipyprotopypes, install with pip::

    pip install ipyprotopypes

or with conda::

    conda install ipyprotopypes
